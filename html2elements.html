HTML ELEMENTS

In HTML (HyperText Markup Language), an element is a building block of a webpage.
It represents the structure and content â€” like headings, paragraphs, images, links, etc.



Basic Structure of an HTML Element:

        <tagname>Content goes here</tagname>
  • <tagname> is the start tag.

  • Content is the data inside.

  • </tagname> is the end tag.


Examples of HTML Elements

1. Heading element

        <h1>This is a heading</h1>

2. Paragraph element

        <p>This is a paragraph.</p>

3. Image element (self-closing)

        <img src="image.jpg" alt="Description">

4. Link element

 
        <a href="https://example.com>Visit Example"</a>

Types of HTML Elements:

Type Example Notes
Block-level <div>,<h1>,<p> Starts on a new line
Inline <span>,<a>,>img> Flows with text
Empty element <br>,<img> No closing tag/content


Common HTML Elements List:

Tag Meaning
<h1> to <h6> Heading(big to small)
<p> Paragraph
<a> Anchor(link)
<img> Image
<br> Line Break
<ul>/<ol>/<li> Lists unordered/ordered/list items
<div> division (section)
<span> Inline container
<form> Form
<input> Form Input
