HTML BASIC TAGS

Tags are used to define elements on a webpage.
Almost all HTML elements have opening and closing tags.



Basic Structure of an HTML Element:

 <tagname>Content goes here</tagname>
  • <tagname> is the start tag.

  • Content is the data inside.

  • </tagname> is the end tag.


Basic HTML Tags

1. Headings: <h1> to <h6>

Use these to create titles or subtitles. There are six levels of headings.

  • <h1> is the largest and boldest.
  • <h6> is the smallest and least bold.

Example:

 <h1>Main Title</h1>
 <h2>Sub Title</h2>
 <h3>Section</h3>
 <h4>Subsection</h4>
 <h5>Note</h5>
 <h6>Smallest Heading</h6>

Output:

Main Title

Sub Title

Section

Subsection

Note
Smallest Heading

2. Paragraphs: <p>

Use this to write a block of text.

Example:

 <p>This is a paragraph of text. It appears with some margin and line spacing.</p>

Output:

This is a paragraph of text. It appears with some margin and line spacing.

3. Line Break: <br>

Used to jump to the next line withput starting a new paragraph. It's a self-closing tag, so it doesn't need </br>

Example:

 <p>Hello Neha!<br>Welcome to HTML learning.</p>

Output:

Hello Neha!
Welcome to HTML learning.

4. Horizontal Line: <hr>

This creates a horizontal line across the page. It used to separate section of content.

Example:

 
 <p>Above the line</p>
 <hr>
 <p>Below the line</p>

Output:

Above the line

Below the line

5. Comments: <!--Comment-->

Comments are invisible notes in your code. They help you or other developers understand the code but don't appear on the webpage.

Example:

 <!--This is a Comment: It won't be shown on the page-->
 
This is visible

Types of HTML Elements:

Type Example Notes
Block-level <div>,<h1>,<p> Starts on a new line
Inline <span>,<a>,<img> Flows with text
Empty element <br>,<img> No closing tag/content


Common HTML Elements List:

Tag Meaning
<h1> to <h6> Heading(big to small)
<p> Paragraph
<a> Anchor(link)
<img> Image
<br> Line Break
<ul>/<ol>/<li> Lists unordered/ordered/list items
<div> division (section)
<span> Inline container
<form> Form
<input> Form Input

Notes:

  • HTML is not case-sensitive,but lowercase is standard
  • Elements can nest inside each other:
    •  <p>This is a <strong>bold</strong>
