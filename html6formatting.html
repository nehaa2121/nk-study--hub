HTML FORMATTING

HTML provides a set of formatting tags that allow you to style and emphasize parts of the text to make content more readable, meaningful, and attractive. These tags are often used tohighlight importance, show deletions or insertions, emphasize text, or display scientific notation.



  1. <b> - Bold
    • Makes the text bold.
    • Does not add sematic meaning.

    html

    <p>This is <b>Bold</b> text.</p>

    Output:

    This is Bold text.



  2. <strong> - Strong Importance
    • Makes the text bold and tells the browsers it's important (sematic meaning).
    • Screen readers emphasize it more.

    html

    <p>This is <strong>important</strong> text.</p>

    Output:

    This is important text.



  3. <i> - Italic
    • Displays text in italics.
    • For names, foreign words, or styled text without emphasis.

    html

    <p>This is <i>italic</i> text.</p>

    Output:

    This is italic text.



  4. <em> - emphasis
    • Emphasizes the text by italicizing it.
    • Also conveys meaning to screen readers.

      • html

      <p>This text is <em>emphasized.</em></p>

      Output:

      This text is emphasized.



  5. <mark> - Highlighted Text

    • Highlights or marks important text with a yellow background (default).

      • html

      <mark>Important message</mark>

      Output:

      Important message


  6. <small>- Smaller Text
    • Displays text in a smaller font.
    • Often used for legal note, disclaimers, or side info.

    html

    <small>This is small text.</small>

    Output:

    This is small text.


  7. <del>- Deleted Text
    • Shows text that has been removed, usually with a strikethrough.

    html

    <del>This offer is expired.</del>

    Output:

    This offer is expired.


  8. <ins> - Inserted Text
    • Shows text that has been added or inserted, usually with an underline.

    html

    <ins>New Date: July 20</ins>

    Output:

    New Date: July 20


  9. <sub> - Subscript
    • Makes the text appear lower (like in chemical formalas).

    html

    H<sub>2</sub>O

    Output:

    H2O


  10. <sup> - Superscript
    • Makes the text appear higher (like in exponents).

    html

    x<sup>2</sup>

    Output:

    x2
