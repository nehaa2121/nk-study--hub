HTML TEXT FORMATTING

Bold Text

1. <b> Tag - Bold (Non - Sematic)

  • Makes text bold, but doesn't add any importance.
  • Used only for styling.

Example:

<p>This is a <b>bold</b> word.</p>

Output:

This is a bold word using <b> tag.

Explanation:

This word "bold" will appear bold just for visual effect.

2. <strong> Tag - Bold (Sematic)

  • Makes text bold and indicates that the text is important.

Example:

<p>Please <strong>read</strong> this carefully</p>

Output:

Please read this carefully.

Explanation:

The word "read" is bold and sematically important (screen readers also emphasize it).
