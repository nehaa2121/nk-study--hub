Introduction to Python

Introduction:

  • Python: Open-source, high-level, interpreted, and general-purpose programming language.
  • Created by: Guido van Possum (1991).
  • Used in: Data science, web development, AI, automation,etc.
  • Why popular:
    • Focuses on solving problems, not syntax.
    • Dynamic typing & automatic memory management.
    • Supports multiple paradigms: Object-oriented, procedural, imperative.
    • Huge standard library and active community.
  • Two main versions are in use today: Python 2 and Python 3 (Python 3 is recommended).




What is Python?

  • Simple & Easy - Clean syntax, easy to read and learn.
  • High-Level & Interpreted - No need for manual memory management or compiling.
  • General Purpose - Used in many fields:
    • Data science
    • Web development
    • Artificial Intelligence (AI)
    • Machine learning
    • Network programming
    • Desktop & database applications
  • Fewer Syntax Rules - Uses punctuation minimally, easy for beginners.
  • Open-Source & Free - Anyone can download and use.
  • Cross-Platform - Works on Windows, Mac, Linux without changes.
  • Extensive Libraries - Comes with built-in modules and third-party packages.


  • Key Features:
    • Interpreted - Code runs line by line at runtime.
    • Interactive - Can test code directly in an interpreter prompt.
    • Object-Oriented - Supports classes and objects for organized programming.
    • Beginner-Friendly - Ideal for new programmers.




Why Learn Python

  • Large number of programming languages exist (C, C++, Java, etc.), but Python is simpler and faster.
  • Uses fewer lines of code for the same task - saves time.
  • Has pre-written libraries and modules for reuse.
  • Good choice for beginners and professionals.
  • Reduces development effort and errors.
  • Can be applied in many areas: web apps, AI, games, scientific computing, data analysis.




Installation of Python

  1. Visit python.org
  2. Go to Downloads - choose the latest version for your OS (e.g., Windows, Mac, Linux).
  3. Download and run the installer.
  4. Tick "Add Python to PATH" and click Install Now.
  5. Verify installation:
    • Open Command Prompt/Terminal - type python version
    • It should display the installed version.
